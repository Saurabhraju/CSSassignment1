  2. CSS Discribes how HTML eliments are to be displayed on screen,paper,or in other media.

  3. CSS is Not neccesary CSS saves lot of work.it can control layout of multiple web pages all at once

CSS can be added in HTML document in 3 ways:

  • Inlin1-by using the style attribute inside HTML element
  • Internal-by using a style element in the head section
  • External-by using a link elememt to link to an external CSS file.
  • The most common way to add CSS, is to keep the styles in external CSS files.However. in this tutorial we will use inline and internal styles, becouse this essier to demonstrate.and easier for you try it yourself.

When comparing CSS class vs ID,the Ifference is that CSS class applies a style to multiple element.Id,on the other hand,applies a style to one unique element.ID is also special in that you a special URL to link directly to an element and it's used by JavaScript.

In CSS,selector are used to target a specific element or range of element on a web page.Once an element has been targeted,a sttyle or set of style can he applied in the element.

There is a wide range of selectors avnilable.Two of the most commonly used are class and ID.Both are used totarget element to which a style should be applied.
